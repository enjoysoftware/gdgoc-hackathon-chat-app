# GDGOC Chat App

Next.js と Firestore を使用した、リアルタイム性に優れたモダンなチャットアプリケーションです。

##  ドキュメント

プロジェクトの詳細については、以下のドキュメントを参照してください。

- [**設計仕様書**](./設計仕様.md)
  - Firestore のデータ構造、コンポーネント構成、ルーティング設計、および Gemini API の利用方法について解説しています。**開発の手を動かす前に必ずお読みください**
- [**セットアップガイド**](./セットアップ方法.md)
  - Firebase プロジェクトの作成、Authentication（Google ログイン）の設定、環境変数の配置、およびローカル環境での起動手順をまとめています。

##  主な機能

- **リアルタイムチャット**: Firestore を利用した即時メッセージ同期。
- **Google 認証**: 安全で簡単なユーザーログイン。
- **AI 連携**: Gemini API を活用したテキスト生成機能。
- **レスポンシブデザイン**: ダークモードを基調とした、使いやすい UI/UX。
- **動的ルーティング**: チャンネルごとのメッセージ管理。

##  クイックスタート

詳細な手順は[セットアップガイド](./セットアップ方法.md)を確認してください。

1. **依存関係のインストール**
   ```bash
   npm install
   ```

2. **環境変数の設定**
   `.env.local` を作成し、Firebase と Gemini の API キーを設定します。

3. **開発サーバーの起動**
   ```bash
   npm run dev
   ```
   ブラウザで `http://localhost:3000` を開きます。
