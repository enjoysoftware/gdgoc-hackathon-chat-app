# チャットアプリ セットアップガイド

この手順に従うことで、提供されたソースコードをベースに、リアルタイムチャット機能を有効化できます。

## 1. Firestore ドキュメントツリー設計

このアプリでは、チャンネルごとにメッセージ履歴を分離するため、以下のサブコレクション構造を採用しています。

```text
/channels (col)
  └ {channelId} (doc) - "project-alpha", "development" など
      └ /messages (col) - 各チャンネル固有のメッセージ
          └ {messageId} (doc)
              ├ text: string       (メッセージ本文)
              ├ sender: string     (送信者名: "Me" など)
              ├ isMe: boolean      (自分が送信したかどうか)
              └ timestamp: serverTimestamp (送信日時)
```

URLが `http://localhost:3000/development` の場合、Firestoreの `/channels/development/messages` からデータを取得します。

## 2. Firebase プロジェクトの作成

1. [Firebase Console](https://console.firebase.google.com/) にアクセスし、「プロジェクトを追加」をクリックします。
2. プロジェクト名（例: `gdgoc-chat-app`）を入力し、作成を完了させます（Google アナリティクスは任意）。

## 3. Authentication の有効化
1. コンソール左メニューの「ビルド」>「Authentication」を選択します。
2. 「始める」をクリックします。
3. 「Sign-in method」タブで「Google」を有効にします。
4. プロジェクトのサポートメールを選択して保存します。

## 3. Firebase 構成情報の取得

1. プロジェクトの概要ページで、中央の **ウェブアイコン (</>)** をクリックしてアプリを登録します。
2. アプリのニックネームを入力し「アプリを登録」をクリックします。
3. 表示された `firebaseConfig` オブジェクトの内容をメモします（後で `.env.local` に使用します）。

## 4. ローカル環境の設定

プロジェクトのルートディレクトリで以下の操作を行います。

### 1. 依存ライブラリのインストール
ターミナルで以下を実行します。
```bash
npm install firebase react-firebase-hooks lucide-react
```

### 2. 環境変数の設定
ルートディレクトリに `.env.local` ファイルを作成し、手順3で取得した値を貼り付けます。

```env
NEXT_PUBLIC_FIREBASE_API_KEY=あなたのAPIキー
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=あなたのプロジェクトID.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=あなたのプロジェクトID
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=あなたのプロジェクトID.appspot.com
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=あなたの送信者ID
NEXT_PUBLIC_FIREBASE_APP_ID=あなたのアプリID
```

## 5. アプリの起動と確認

1. 以下のコマンドで開発サーバーを起動します。
   ```bash
   npm run dev
   ```
2. ブラウザで `http://localhost:3000` を開きます。
3. 下部の入力エリアにメッセージを入力し、Enterキーまたは送信ボタンを押します。
4. Firebase Console の Firestore 画面を開き、`messages` コレクションが自動作成され、データが保存されていることを確認してください。

---

### 補足：セキュリティルールについて（推奨）
「テストモード」の期限が切れる前、または開発が進んだら、Firestore の「ルール」タブを以下のように更新して、データの読み書きを許可してください。

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /messages/{messageId} {
      allow read, write: if true; // 開発用：誰でも読み書き可能
    }
  }
}
```
